The Larp Circle
Statement

A group of n dedicated larpers showed up to an outdoor event. Half were larping Chainsaw Man characters, a few were doing a full Subahibi visual novel reenactment, and the rest were arguing about whether Subaru from Re:Zero would survive their obstacle course.

Eventually they agreed to do round dances. Every person has exactly two neighbors. If only 2 people are in a dance, they share the same neighbor on both sides.

Everyone was so deep in character that each person only remembered ONE of their two neighbors. Given who each person remembered, determine the minimum and maximum number of separate round dances that could have taken place.

Subaru tried Return by Death to remember both neighbors. Still only got one.

Input

The first line contains a single integer t (1 ≤ t ≤ 104) -- the number of test cases.

Each test case begins with a single integer n. The next line contains n integers a1, a2, ..., an, where ai is the one neighbor person i remembers.

Output

For each test case, print two integers on one line: the minimum and maximum number of round dances.

Constraints
1 ≤ t ≤ 104
2 ≤ n ≤ 2×105
1 ≤ ai ≤ n, ai ≠ i
Sum of n across all test cases ≤ 2×105
Input guaranteed to correspond to at least one valid arrangement.
Sample I/O
Sample 1
Input
3 6 2 1 4 3 6 5 6 2 3 1 5 6 4 2 2 1
Output
1 3 2 2 1 1
