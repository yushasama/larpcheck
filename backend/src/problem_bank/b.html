Tung Tung Tung Sahur vs Larp Larp Larp Sahur
Statement

The village of Sahur is under attack. The villagers have lined up n warriors in order of battle power (sorted ascending). Tung Tung Tung Sahur needs to find if a warrior with power level k exists in the lineup, but he can only bang his drum O(log n) times before Larp Larp Larp Sahur wakes up and ruins everything.

Find the 0-indexed position of the warrior with power k. If no such warrior exists, return -1.

You MUST solve this in O(log n). Linear scan = Larp Larp wins.

Input

The first line contains a single integer t (1 ≤ t ≤ 104), the number of test cases.

Each test case consists of two lines. The first line contains two integers n and k. The second line contains n space-separated integers forming the sorted array.

Output

For each test case, print a single integer: the 0-indexed position of k, or -1 if not found.

Constraints
1 ≤ t ≤ 104
1 ≤ n ≤ 104
-104 < nums[i], k < 104
All elements unique. Array sorted ascending.
Array is 0-indexed
Sample I/O
Sample 1
Input
2 6 9 -1 0 3 5 9 12 6 2 -1 0 3 5 9 12
Output
4 -1
