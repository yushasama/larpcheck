Super Creek's Training Ground
Statement

Super Creek has built a training course shaped like a tree of n checkpoints rooted at checkpoint 1. Each checkpoint has a stamina value.

Goldship keeps asking: "if I ran through every checkpoint in your subtree starting from u, how much total stamina would I need?" Answer each of Goldship's q questions. She has a 0.3 second attention span so you better be fast.

Input

The first line contains a single integer t (1 ≤ t ≤ 104) -- the number of test cases.

Each test case begins with two integers n and q. The next line contains n integers s1, s2, ..., sn -- the stamina values. The next n-1 lines each contain two integers u and v describing an edge. The next q lines each contain a single integer u -- a query node.

Output

For each query, print the total stamina sum over the subtree of u.

Constraints
1 ≤ t ≤ 104
1 ≤ n, q ≤ 2×105
1 ≤ si ≤ 109
Sum of n and sum of q across all test cases ≤ 2×105
Sample I/O
Sample 1
Input
1 5 3 1 2 3 4 5 1 2 1 3 2 4 2 5 2 1 4
Output
11 15 4
