Special Week's Training Log
Statement

Special Week has just arrived at Tracen Academy and needs to log her training sessions. Her trainer gives her the following rules:

If the session number is divisible by both 3 and 5, write UmaMusume.
If divisible by 3 only, write Uma.
If divisible by 5 only, write Musume.
Otherwise, write the session number itself.

Given n, print the full training log from session 1 to n.

Input

The first line contains a single integer t (1 ≤ t ≤ 104), the number of test cases.

Each test case contains a single integer n.

Output

For each test case, print n lines according to the rules above.

Constraints
1 ≤ t ≤ 104
1 ≤ n ≤ 104
Sessions are numbered 1 to n (1-indexed)
Sample I/O
Sample 1
Input
1 15
Output
1 2 Uma 4 Musume Uma 7 8 Uma Musume 11 Uma 13 14 UmaMusume
