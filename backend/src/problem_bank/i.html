Ayanokoji's Exposure Protocol
Statement

n people in a group chat all set their pfp to Ayanokoji after watching Classroom of the Elite. Each one claims to have understood "the true meaning" of COTE. Ayanokoji, who planned this from the start, has prepared exactly n quiz topics, one per person.

He already knows each person's exposure score for each topic: how badly they get caught larping if quizzed on it. Being Ayanokoji, he wants to assign topics to maximize total exposure. Being you, you want to predict his optimal assignment to minimize your own damage.

Assign exactly one topic to each person such that no two people share a topic. Find the assignment that minimizes total exposure score.

Input

The first line contains a single integer t (1 ≤ t ≤ 10), the number of test cases.

Each test case begins with a single integer n, the number of people and topics. The next n lines each contain n integers, where e[i][j] is the exposure score of person i if assigned topic j.

Output

For each test case, print a single integer: the minimum total exposure score over all valid assignments.

Constraints
1 ≤ t ≤ 10
1 ≤ n ≤ 18
0 ≤ e[i][j] ≤ 106
People and topics are numbered 0 to n-1 (0-indexed)
Each person assigned exactly one topic, each topic assigned to exactly one person
Sample I/O
Sample 1
Input
1 3 9 2 7 3 6 1 1 8 5
Output
8
Sample 2
Input
1 2 5 3 2 8
Output
10
Sample 1: assign person 0 to topic 1 (cost 2), person 1 to topic 2 (cost 1), person 2 to topic 0 (cost 1). Total = 4... wait, recheck: 2+1+5=8. Ayanokoji already knew. He always knows.
