Nokotan's Antler Count
Statement

Noko Shikanoko, known to her friends as Nokotan, is counting antler points across n deer in a forest. Given q questions of the form "how many total antler points do deer 1 through i have?", answer each one.

Input

The first line contains a single integer t (1 ≤ t ≤ 104), the number of test cases.

Each test case begins with two integers n and q. The next line contains n integers a1, a2, ..., an. The next q lines each contain a single integer i.

Output

For each query, print the sum of a1 through ai on its own line.

Constraints
1 ≤ t ≤ 104
1 ≤ n, q ≤ 2×105
1 ≤ ai ≤ 109
1 ≤ i ≤ n for each query (1-indexed)
Sum of n and sum of q across all test cases ≤ 2×105
Sample I/O
Sample 1
Input
1 5 3 1 3 2 5 4 1 3 5
Output
1 6 15
